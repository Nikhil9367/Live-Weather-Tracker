# 🌦️ Live Weather Tracker

A simple and responsive **Live Weather Tracker** web application that allows users to check real-time weather information of any city. The app fetches live data from a weather API and displays temperature, humidity, wind speed, and weather conditions.

---

## 🚀 Features

- 🌍 Search weather by city name  
- 🌡️ Real-time temperature display  
- 💨 Wind speed information  
- 💧 Humidity level  
- 🌥️ Weather condition icons (Cloudy, Sunny, Rainy, etc.)  
- 📱 Fully responsive design  
- ⚡ Fast and lightweight  

---

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript  
- **API:** OpenWeatherMap API  
- **Styling:** CSS

## ⚙️ Installation & Setup

1. Clone the repository:
```bash
git clone https://github.com/your-username/live-weather-tracker.git
cd live-weather-tracker
```

2. Open the project:
- Open `index.html` in your browser  

---

## 🔑 API Setup

1. Go to https://openweathermap.org/  
2. Create an account  
3. Generate an API key  
4. Replace it in `script.js`:

```javascript
const apiKey = "YOUR_API_KEY";
```

---

## 🧠 How It Works

- User enters a city name  
- The app sends a request to the weather API  
- API returns data in JSON format  
- Data is displayed dynamically on the webpage
